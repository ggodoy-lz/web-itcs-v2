# Identidad Visual y Branding de iTCS S.A.

## Paleta de Colores Corporativa

### Colores de Base (Interfaz Oscura)
- **Fondo Principal**: `#020617` (Slate 950) — Se utiliza como fondo base de todo el sitio.
- **Fondo de Cards/Secciones**: `#0f172a` (Slate 900) — Se utiliza para dar profundidad a elementos de contenido.
- **Bordes y Divisores**: `rgba(30, 41, 59, 0.4)` (Slate 800 con opacidad).

### Acentos de Marca (Azul iTCS)
- **Principal**: `#2563eb` (Blue 600) — Color de botones primarios y acciones principales.
- **Destacado**: `#60a5fa` (Blue 400) — Textos secundarios, iconos y hover de enlaces.
- **Glow/Brillo**: `rgba(59, 130, 246, 0.08)` — Sombra sutil utilizada en cards interactivas.

### Paleta Semántica por Servicio
Cada área técnica tiene una identidad cromática propia utilizada en badges y bullets:
- **Azul**: Infraestructura & Productividad.
- **Rojo**: Ciberseguridad.
- **Verde**: Networking & Telecomunicaciones.
- **Amarillo**: Base de Datos.
- **Púrpura**: Desarrollo de Sistemas.
- **Cian**: Respaldo & Continuidad.
- **Naranja**: Cómputo de Misión Crítica.
- **Teal (Azul Verdoso)**: Soporte Técnico.
- **Rosado**: Obras Civiles.

### Gradientes Utilizados
- **Texto con Gradiente**: De `#60a5fa` (Blue 400) a `#818cf8` (Indigo 400) terminando en `#34d399` (Emerald 400).
- **Gradiente Hero**: Fondo dinámico de `#020617` a `#1e1b4b` (Indigo 950) sutil.

## Tipografía Corporativa

### Fuente Principal
- **Familia**: **Inter** (Google Fonts).
- **Variables Alternativas**: Plus Jakarta Sans, Outfit, Space Grotesk (configuradas como alternativas en el sistema).
- **Uso**: Sans-serif moderno para legibilidad en pantallas técnicas.

### Jerarquía
- **H1/H2**: Pesos ExtraBold (800) o Black (900) para impacto corporativo.
- **Cuerpo**: Peso Regular (400) con interletrado suavizado (antialiased).

---

## Recursos Visuales
- **Patrón de Fondo**: Grilla técnica (`hero-grid`) de 60px x 60px con líneas de color Indigo al 7% de opacidad.
- **Iconografía**: Set de iconos de línea fina (`Heroicons/outline`) en tamaño 24px-28px.
- **Logotipos**: Versión en blanco con transparencia para fondos oscuros (`.webp`).
